# letsACE

Certification practice platform built with React, FastAPI, and PostgreSQL.

## Features
- Secure user authentication with JWT
- Custom quiz engine
- Performance dashboard
- Question review system
- Admin question management
- Responsive UI

## Tech Stack

Frontend:
React
Vite
Axios
React Router

Backend:
FastAPI
SQLAlchemy
PostgreSQL
Alembic
JWT Authentication

Deployment:
Vercel
Render
